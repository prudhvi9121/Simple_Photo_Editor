

<h1>Simple Photo Editor</h1>
<p>A web-based photo editor built with HTML5, CSS, and JavaScript that allows users to upload an image, crop, resize, apply filters, reset, and download the edited image.</p>

<h2>Features</h2>
<ul>
    <li><strong>Upload Image:</strong> Upload an image from your local device.</li>
    <li><strong>Crop Image:</strong> Crop the image to a central part of the canvas.</li>
    <li><strong>Resize Image:</strong> Resize the image to specified dimensions.</li>
    <li><strong>Apply Filters:</strong> Apply various filters such as grayscale, sepia, blur, brightness, and contrast.</li>
    <li><strong>Reset Image:</strong> Reset the image to its original state.</li>
    <li><strong>Download Image:</strong> Download the edited image in PNG format.</li>
</ul>

<h2>Technologies Used</h2>
<ul>
    <li>HTML5</li>
    <li>CSS3</li>
    <li>JavaScript</li>
</ul>
